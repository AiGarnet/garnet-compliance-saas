Garnet AI - Vendor Onboarding Platform

A comprehensive vendor compliance and onboarding platform with a modern UI built with Next.js and Tailwind CSS.

Project Structure

Frontend

  • Next.js frontend/ - Next.js application with Tailwind CSS
  • frontend/app/ - Next.js app directory structure
  • frontend/components/ - Reusable UI components
  • frontend/lib/ - Utility functions and shared code

Backend

  • Express.js backend/ - Express.js API server
  • backend/src/ - Backend source code
  • backend/src/utils/ - Utility functions
  • backend/src/services/ - Service layer

Chatbot

  • Flask chatbot/ - Flask-based AI chatbot microservice
  • chatbot/server/ - Server implementation

Other

  • scripts/ - Utility scripts
  • public/ - Static assets
  • .github/workflows/ - CI/CD configuration

Getting Started

Prerequisites

  • Node.js (v18 or higher)
  • npm or yarn
  • Docker and Docker Compose (for containerized setup)

Installation

  1. Install frontend dependencies: cd frontend && npm install
  2. Install backend dependencies: cd backend && npm install
  3. Install chatbot dependencies: cd chatbot && pip install -r requirements.txt

Development

  • Start frontend: cd frontend && npm run dev
  • Start backend: cd backend && npm run dev
  • Start chatbot: cd chatbot && python app.py
